# Indian Penal Code (IPC) Foundations

The Indian Penal Code (IPC) is India's principal criminal code for substantive offences.
It was drafted by the First Law Commission and came into force in 1862.

The Code contains 511 sections arranged into 23 chapters.
It defines offences, legal ingredients, general exceptions, and punishments.

Key foundational ideas:
- Mens rea (mental element) and actus reus (physical act) are central to criminal liability.
- Some provisions create joint liability, such as Section 34 (common intention) and Section 149 (unlawful assembly).
- General exceptions (Sections 76 to 106) include private defence and acts done by mistake of fact in certain conditions.
